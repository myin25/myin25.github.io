Multi-Module Brain Injury Triage System

Independent Research Project

Developed a comprehensive machine learning system for brain injury triage that combines clinical metrics with medical imaging analysis. The system employs multiple ML algorithms including Logistic Regression, Random Forest, Gradient Boosted Descent, and Convolutional Neural Networks with hybrid data models to provide accurate, rapid assessment of traumatic brain injuries.

  • 99.28% accuracy achieved on PECARN (Pediatric Emergency Care Applied Research Network) clinical metrics dataset
  • 91.25% accuracy on CT scan image analysis, enabling automated radiological assessment
  • Implemented hybrid approach combining structured clinical data with image-based deep learning for robust predictions
  • Designed system with potential for real-world deployment in emergency department settings
